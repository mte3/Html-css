# networth

网值头条app

# 文件夹结构
    jh_net_worth_headlines

        -android 安卓原生代码

        -assets 静态文件，图片，图标，字体等资源

        -ios ios原生代码

        -lib dart代码

            -components  自定义组件

            -dto  后台数据交换实体

            -model 前端自定义数据模型

            -pages 页面

            -utils 工具类

        -plugins 外部插件包

        -test 测试

       -pubspec.yaml 配置文件
